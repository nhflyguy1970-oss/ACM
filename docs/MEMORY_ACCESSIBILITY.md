**Status:** Canonical for M10 (implemented)

## Distinction

| State | Meaning |
|-------|---------|
| Memory exists | Artifact still in the cognitive store (Experience immutable; Concept/Association living) |
| Memory is accessible | Structure participates easily in Activation / Remembering |

Human-like failure is usually **accessibility change**, not disappearance.

## Stages

```
Highly Accessible → Accessible → Less Accessible → Dormant
  → Rarely Activated → Archived → Eligible for user-approved pruning
```

Experiences remain immutable at every stage. Soft cool only.

## Reactivation

Strong cues, identity adjacency, powerful associations, repeated exposure, reflection, learning, offline replay, context and attention shifts can raise accessibility again.
