# Memory Recombination — ACM

**Status:** Canonical for M13 (implemented)  
**Cognitive question:** *What new memories can emerge by recombining existing memories?*  
**Companions:** [`CREATIVITY_FOUNDATIONS.md`](CREATIVITY_FOUNDATIONS.md) · [`MENTAL_SIMULATION.md`](MENTAL_SIMULATION.md) · [`BIOLOGICAL_VS_TECHNICAL_FUNCTION.md`](BIOLOGICAL_VS_TECHNICAL_FUNCTION.md)

## Biological function

Humans recombine episodic and semantic fragments into novel scenes and ideas (constructive memory / generative episodic recombination — **well-supported**). This supports creativity and flexible transfer without erasing originals. Mechanisms of “how novelty is scored” remain partly **emerging**.

## Technical function

| Field | Content |

| Organ | `RecombinationOrgan` |
| Inputs | Activation field, Concepts, Associations, Experiences (read-only), Priority, Simulation/Prediction hints |

## Ownership

| Owns | Does not own |
|------|----------------|
| Novel combinations of existing fragments | Structural analogy mapping (M14) |
| Cross-domain / pattern / concept / association recombination | Planning, decisions, executive reasoning |
| Temporary recombined structures | Rewriting Experiences |

## Distinguishes from Simulation (M12)

| Simulation | Recombination |
|------------|---------------|
| “What futures might follow?” (prospective path) | “What new blend can memory form?” (combinatorial novelty) |
| Ordered hypothetical steps | Cross-fragment blend with novelty score |

## Intentional omissions

Unconstrained generative language, artistic style engines, planning of actions.
