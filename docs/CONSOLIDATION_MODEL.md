## Replay

Offline Cognition selects recent Reflective Experiences and high `access_count` Concepts, re-activates neighborhoods via the shared Activation Architecture (read-only energy), and issues low-impact Learning adaptations with `sleep_batch_id`.

## Memory stabilization

Coherent structures (strong Associations, multi-evidence Concepts) receive small reinforcing Adaptations. Contradictory Reflective outcomes receive weakening Adaptations — history unchanged.

## Pattern reinforcement

Repeated labels / co-activation clusters get preferential reinforcement (spacing/offline practice — well-supported behaviorally; implemented functionally).

## Confidence stabilization

Online Learning may nudge confidence; Offline passes re-average extremes toward evidence mass (stabilize, not invent certainty).

## Offline organization

- Dormancy of very weak Associations (accessibility cool)  
- Merge/alias **proposals** only  
- Salience overlay soft-touch on high-traffic Experiences (current overlays only — birth frozen)

Never fabricate Experiences or Reflective events about events that did not occur.
