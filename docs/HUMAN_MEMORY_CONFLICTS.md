# Human Memory Conflicts — ACM

**Status:** Canonical scientific companion for M15  
**Companions:** [`MEMORY_RECONCILIATION.md`](MEMORY_RECONCILIATION.md) · [`EVIDENCE_AND_CORROBORATION.md`](EVIDENCE_AND_CORROBORATION.md)

## Well-supported

- Autobiographical memory is constructive; contradictions are common.
- Source monitoring distinguishes *what* from *how I know*.
- Competing memories can remain available; context selects retrieval.
- Reconsolidation updates living representations without requiring literal overwrite of all earlier traces.

## Emerging

- Exact weightings of social vs perceptual corroboration.
- How sleep/offline cognition reshapes conflict without inventing history (handled elsewhere in ACM Offline Cognition; Reconciliation remains online lineage).

## Speculation (not modeled as fact)

- Single winner “global workspace” of truth for every conflict.

## ACM engineering interpretation

| Human pattern | ACM response |
|---------------|--------------|
| Retain both episodes | Experiences immutable |
| Living summary changes | Concept confidence / attributes evolve |
| Explain why conflict exists | `ReconciliationRecord.factors` + summary |
| Context-dependent truth | `context_dependent` status |
| Feel less certain when conflicted | Confidence organ recalibration |

## What ACM does not model

Jury deliberation for moral truth; courtroom evidence rules; executive selection of which action to take under conflict.
