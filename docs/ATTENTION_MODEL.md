## Definition

Attention allocates finite cognitive resource toward encoding, cueing, remembering, reflection, learning, and offline replay. It does **not** plan, decide goals, or reason about the world.

## Ownership

| Owns | Does not own |
|------|----------------|
| Encoding gate weight | Goal lifecycle (Goals bias Attention) |
| Cue / recall attention weight | Concept meaning |
| Replay selection bias | Deletion / hard erase |
| Priority investment signals | Planning / prediction |

## Emergence

Priority and attention intensity emerge from Identity, Goals, Novelty, Importance, Repetition, Context, Conflict, Reflection, Learning, Offline batches, Confidence, and Salience — recorded as **observable factor pulls**, not a fixed secret weight table for living investment.

Base class labels (`AttentionClass`) remain explainable classifiers; living **priority** on Concepts evolves through experience (P6 / P14).

## Activation

Attention modulates energy into the **one** Cognitive Activation Architecture. No second activation engine.
