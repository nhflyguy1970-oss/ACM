## Purpose

Reflection is ACM’s first **metacognitive** organ. It evaluates reconstructed cognition. It does not search, retrieve, learn, predict, plan, or rewrite history.

## Ownership

| Owns | Must never assume |
|------|-------------------|
| Evaluation of Remembering reconstructions | History rewrite |
| Contradiction / consistency observation | Concept meaning formation |
| Confidence assessment (as evaluation) | Silent Concept/Association mutation |
| Question & hypothesis *generation* (as artifacts) | Learning / adaptation |
| Pattern observation over activations | Decision / planning |
| Birth of Reflective Experiences | Behavior modification |

### Depends upon

Identity · Experiences · Concepts · Associations · Remembering · Goals · Attention · Working Memory · Context · **shared Activation Architecture** (via Remembering)

### Future organs that depend upon Reflection

Learning · Prediction · Planning · Creativity · Metacognition maturity · assent-gated self-improvement

## Evaluation

Given a cue, Reflection:

1. Obtains a Reconstruction from Remembering (Activation Architecture reused — no second activation model)  
2. Evaluates confidence, ambiguity, evidence mass, competing recollections, activation breadth  
3. Emits observable outcomes (honest uncertainty allowed)  
4. Births a **Reflective Experience** linked via `reflects_on` lineage  

## Reflective Experiences

Every organ reflection leaves immutable history. See [`REFLECTIVE_EXPERIENCES.md`](REFLECTIVE_EXPERIENCES.md).

## Outcomes (observable, template-class)

| Outcome | Meaning |
|---------|---------|
| `sufficient` | Evidence adequate for stated recollection |
| `insufficient_evidence` | Too little grounding |
| `uncertainty` | Low confidence / honest don’t-know |
| `contradiction` | Competing recollections or conflict signals |
| `consistency` | Stable single reconstruction |
| `pattern` | Neighborhood / co-activation pattern noted |
| `question` | Follow-up question artifact |
| `hypothesis` | Provisional interpretation (not a decision) |
| `insight` | Condensed evaluative summary |

Intellectual honesty: *I don’t know*, *conflicting evidence*, *multiple interpretations* are first-class.

## Relationship to Learning

Reflection prepares observations. **Learning** (M7) consumes Reflective Experiences to adapt living structures. Reflection itself does not adapt ACM. Architecture/policy change remains assent-gated.
