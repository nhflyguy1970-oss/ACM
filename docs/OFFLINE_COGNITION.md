## Purpose

Offline Cognition is sleep-like **memory reorganization**. It does not sense the world. It works only on existing cognitive artifacts.

## Owns

| Owns | Must never assume |
|------|-------------------|
| Replay of reflective / high-access structures | Encoding new external Experiences |
| Consolidation batches | Inventing history |
| Abstraction proposals (merge candidates) | Applying high-impact merges silently |
| Association/Concept refinement via Learning hooks | Learning’s online adaptation ownership alone |
| Confidence / salience stabilization passes | Architectural self-modification |
| Identity stabilization (low-impact) | Conflicting identity flips |

## Functional sleep (not biological simulation)

Replay reinforces coherent Concepts/Associations through Learning Adaptations marked with a sleep batch id. Weak edges may cool (dormancy) without deleting Experiences. Duplicate-label merges remain **proposals**.

## Depends upon

Learning · Concepts · Associations · Experiences · Identity · Reflection artifacts · Activation (for neighborhood selection)
