## Purpose

Describe how Concepts form **cognitive neighborhoods** — not arbitrary graphs.

## Neighborhoods

From a Concept, the Association organ yields:

- Immediate neighbors (high forward strength)  
- Near ring (moderate strength / shared context)  
- Far ring (weak traversable links)  

Neighborhoods are meaning-shaped: family resemblance, co-experience, goals, and identity push membership — not node-degree vanity metrics.

## Clusters / communities / domains

Repeated local density produces:

| Structure | Meaning |
|-----------|---------|
| **Cluster** | Tightly co-activated Concepts |
| **Community** | Softly separated dense regions |
| **Domain / theme** | Shared context or goal coloring |

M4 exposes observational cluster summaries (connected components / density hints). Full community detection algorithms may evolve later without redesigning Association identity.

## Emergent organization

Networks reorganize as strengths change. Dormant links drop from default neighborhoods without destroying history. New Experiences can open unexpected bridges (creative distance).

## Future activation & traversal

Future Remembering should:

1. Activate seed Concepts  
2. Spread along directional Association strengths  
3. Respect distance bands and context  

M4 builds the substrate for that spread — it does not run a rememberer.
