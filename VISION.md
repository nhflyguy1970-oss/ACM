# Vision — ACM

ACM (Aria Cognitive Memory) is a **standalone cognitive memory engine**: the closest practical approximation of useful human memory behavior that can exist in software.

It is **not** Aria. Aria will be the first *consumer* after ACM matures. The engine must serve any future agent — assistants, robotics, research systems, domain specialists — without assuming any particular host.

## What we are building

A practical cognitive architecture in which:

- Experiences become Concepts
- Concepts form Associations
- Goals and Attention shape encoding and recall
- Remembering changes memory (reconsolidation)
- Sleep consolidates, prunes, and proposes structure
- Identity and metacognition emerge from interacting systems over time
- Multimodal envelopes are first-class, never second-class to text

## What we are not building

- Artificial consciousness or simulated sentience
- Biological or neuroscience imitation for its own sake
- Scripted persona / hard-coded “self”
- A host-specific memory database wrapped as cognition
- Prompt dumps or chain-of-thought as observability

## North-star outcome

Any agent can install ACM, open a `CognitiveEngine`, and grow durable, contextual, goal-aware memory with measurable cognitive observables — without importing Aria, Mission Control, or any other host stack.
